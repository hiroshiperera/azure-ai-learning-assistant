# Azure Foundations

## What is an Azure Subscription?

An Azure Subscription is a billing and management container that allows you to create and manage Azure resources.

Without a subscription, Azure resources cannot be created.

---

## What is a Resource Group?

A Resource Group is a logical container used to organize Azure resources.

Example:

Subscription

↓

Resource Group

↓

Azure OpenAI

↓

Storage

↓

AI Search

The Resource Group itself is free.

---

## Why use Resource Groups?

- Organize resources
- Easier cost management
- Easier permissions
- Easier deletion of related resources

---

## What is Cost Management?

Azure Cost Management allows monitoring cloud expenses.

Budgets can generate alerts but do not automatically stop resources.
