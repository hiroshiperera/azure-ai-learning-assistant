# Azure Fundamentals

## 📘 About This Module

Welcome to the **Azure Fundamentals** module of my Azure AI Learning Journey.

This module covers the foundational Azure concepts that every Azure AI Engineer should understand before building AI applications. It documents both the theoretical knowledge and the practical implementation required to prepare an Azure environment for AI development.

The concepts learned in this module serve as the foundation for all future Azure AI services, including Azure OpenAI, Azure AI Search, Azure AI Foundry, Document Intelligence, AI Agents, and Retrieval-Augmented Generation (RAG).

This module was created as part of my personal journey toward becoming an Azure AI Engineer and preparing for the **Microsoft AI-102: Designing and Implementing a Microsoft Azure AI Solution** certification.

---

# 🎯 Learning Objectives

After completing this module, I should be able to:

- Understand Microsoft Azure and cloud computing fundamentals.
- Explain the purpose of Azure Subscriptions.
- Understand Azure's Pay-As-You-Go pricing model.
- Create and manage Azure Budgets.
- Explain the purpose of Resource Groups.
- Understand Azure Regions and why they matter.
- Create an Azure OpenAI Resource.
- Navigate the Azure Portal.
- Use Azure AI Foundry.
- Prepare Azure for AI application development.
