# Azure OpenAI

## What is Azure OpenAI?

Azure OpenAI is Microsoft's managed AI platform that provides access to OpenAI foundation models.

---

## Azure OpenAI Resource

The Azure OpenAI Resource is a container that hosts AI model deployments.

---

## Model Deployment

A deployment is a configured instance of a model.

Applications communicate with the deployment name rather than directly with the model.
