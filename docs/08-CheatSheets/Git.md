# Git Cheat Sheet

## Initialize Repository

git init

---

## Check Status

git status

---

## Add Files

git add .

---

## Commit

git commit -m "Initial Commit"

---

## View Log

git log

---

## Create Branch

git checkout -b feature/chatbot

---

## Switch Branch

git checkout main

---

## Push

git push origin main

---

## Pull

git pull origin main

---

## Clone

git clone <repository_url>

---

## View Remote

git remote -v
