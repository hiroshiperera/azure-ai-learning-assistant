
# Python Cheat Sheet

## Create Virtual Environment

python -m venv .venv

---

## Activate

.venv\Scripts\activate

---

## Deactivate

deactivate

---

## Install Packages

pip install -r requirements.txt

---

## Freeze Requirements

pip freeze > requirements.txt

---

## Run Program

python app.py

---

## Upgrade pip

python -m pip install --upgrade pip
